# Neovim 0.12 Config

Minimal, annotated from-scratch config targeting Neovim 0.12.1+.
No distros. No magic. Every line is readable.


## Structure

```
~/.config/nvim/
├── init.lua                  ← entry point, sets load order
├── nvim-pack-lock.json       ← auto-generated by vim.pack (commit this)
└── lua/
    └── config/
        ├── options.lua       ← vim.opt settings
        ├── pack.lua          ← plugin declarations (vim.pack)
        ├── lsp.lua           ← native LSP via vim.lsp.config/enable
        ├── keymaps.lua       ← all keybindings
        └── autocmds.lua      ← autocommands
```

## Installation

```bash
# Backup existing config
mv ~/.config/nvim ~/.config/nvim.bak

# Copy this config
cp -r . ~/.config/nvim

# Install dependencies
./install-deps.sh

# Open nvim — it will bootstrap plugins on first launch
nvim

# Then install plugins
:Pack install
```

## Checking LSP status

```
:checkhealth vim.lsp    ← see which servers are configured and active
:lsp                    ← manage running clients
:Mason                  ← manage installed servers/tools
```
